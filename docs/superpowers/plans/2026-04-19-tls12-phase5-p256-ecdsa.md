# TLS 1.2 Phase 5: P-256 + ECDSA Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现 P-256 (secp256r1) 曲线运算 + ECDSA-SHA256 签名验证，使 viv 能验证 ECDSA 签发的证书与 TLS 1.2 ServerKeyExchange 签名。

**Architecture:** `FieldElement` 基于 `[u64; 4]` 定宽 limb，模 p 的加减乘用 BigUint 协助（mul + div_rem 做约减；比 Solinas 多一点开销但更简单正确）。`Point` 用 Jacobian 坐标 (X:Y:Z)，标量乘用朴素 double-and-add。`EcdsaPublicKey::from_spki` 解 SPKI，`verify_ecdsa_sha256` 做完整验证。

**Tech Stack:** Rust 2024；依赖 Phase 1 `core::bigint::BigUint`（含新加的 `mod_inverse`）、Phase 2 `core::asn1::Parser`、已有 `core::net::tls::crypto::sha256::Sha256`；零第三方 crate。

**Spec reference:** `docs/superpowers/specs/2026-04-19-tls12-phase5-p256-ecdsa-design.md`

**与 spec 的偏差说明**：Spec 提到 Solinas quick reduction，plan 改用 **BigUint 通用约减**（`mul → div_rem(p)`）。性能估算仍远低于 200ms 验证上限；省 ~150 LOC 的位移代码且避免 bug 面。API 完全一致。

```rust
    /// Modular inverse via the extended Euclidean algorithm using signed
    /// arithmetic (BigUint + bool sign tag internally).
    ///
    /// Returns `Some(x)` such that `(self * x) mod modulus == 1`, or
    /// `None` when `gcd(self, modulus) != 1` or `modulus` is zero.
    pub fn mod_inverse(&self, modulus: &Self) -> Option<Self> {
        if modulus.is_zero() {
            return None;
        }
        if modulus == &Self::one() {
            // Every integer is 0 mod 1; conventional inverse is 0 but
            // formally undefined. Return None to match the "no inverse"
            // semantics callers expect.
            return None;
        }

        // Reduce self mod modulus first so a fits in [0, modulus).
        let (_, a0) = self.div_rem(modulus)?;
        if a0.is_zero() {
            return None;
        }

        // Extended Euclidean with signed coefficients represented as
        // (magnitude, sign). sign=false means positive, true means negative.
        //   a * s + modulus * t = gcd(a, modulus)
        // We track (old_r, r) and (old_s, s); t is not needed.
        let mut old_r = a0;
        let mut r = modulus.clone();
        let mut old_s: (BigUint, bool) = (Self::one(), false);
        let mut s: (BigUint, bool) = (Self::zero(), false);

        while !r.is_zero() {
            let (q, new_r) = old_r.div_rem(&r)?;
            // old_r := r; r := old_r - q*r  (= new_r)
            old_r = r;
            r = new_r;
            // old_s, s := s, old_s - q*s   (in signed arithmetic)
            let q_s = signed_mul(&q, &s);
            let next_s = signed_sub(&old_s, &q_s);
            old_s = s;
            s = next_s;
        }

        // Now old_r = gcd(a, modulus). If it's not 1, no inverse exists.
        if old_r != Self::one() {
            return None;
        }

        // old_s is the inverse in signed form. Reduce mod modulus to [0, n).
        if old_s.1 {
            // negative: inverse = modulus - (|old_s| mod modulus)
            let (_, r) = old_s.0.div_rem(modulus)?;
            if r.is_zero() {
                Some(Self::zero())
            } else {
                modulus.checked_sub(&r)
            }
        } else {
            let (_, r) = old_s.0.div_rem(modulus)?;
            Some(r)
        }
    }
```

在文件底部加辅助函数（`impl` 外）：

```rust
/// Signed BigUint multiply: (|a|, _) * (|b|, sign_b) = (|a|*|b|, sign_b).
/// Used by mod_inverse only; not exposed.
fn signed_mul(a: &BigUint, b: &(BigUint, bool)) -> (BigUint, bool) {
    (a.mul(&b.0), b.1)
}

/// Signed BigUint subtract: (a_val, a_sign) - (b_val, b_sign).
fn signed_sub(a: &(BigUint, bool), b: &(BigUint, bool)) -> (BigUint, bool) {
    match (a.1, b.1) {
        (false, false) => {
            // a - b
            if a.0.cmp(&b.0) != std::cmp::Ordering::Less {
                (a.0.checked_sub(&b.0).unwrap_or(BigUint::zero()), false)
            } else {
                (b.0.checked_sub(&a.0).unwrap_or(BigUint::zero()), true)
            }
        }
        (false, true) => {
            // a - (-b) = a + b
            (a.0.add(&b.0), false)
        }
        (true, false) => {
            // (-a) - b = -(a + b)
            (a.0.add(&b.0), true)
        }
        (true, true) => {
            // (-a) - (-b) = b - a
            if b.0.cmp(&a.0) != std::cmp::Ordering::Less {
                (b.0.checked_sub(&a.0).unwrap_or(BigUint::zero()), false)
            } else {
                (a.0.checked_sub(&b.0).unwrap_or(BigUint::zero()), true)
            }
        }
    }
}
```

注意 `checked_sub().unwrap_or(zero())` 在 signed_sub 里是逻辑保证不触发的分支（由 cmp 前置），但为避免 `unwrap` 用 `unwrap_or` 兜底。签名函数不是 `src/` 风格违规（目标文件是 `src/core/bigint.rs`，但 `.unwrap_or` 不是 `.unwrap`，允许）。

```rust
impl FieldElement {
    pub fn zero() -> Self {
        FieldElement(BigUint::zero())
    }

    pub fn one() -> Self {
        FieldElement(BigUint::one())
    }

    /// Parse 32-byte big-endian value. Returns `None` if ≥ p.
    pub fn from_bytes_be(bytes: &[u8; 32]) -> Option<Self> {
        let n = BigUint::from_bytes_be(bytes);
        if n.cmp(&p_modulus()) != std::cmp::Ordering::Less {
            return None;
        }
        Some(FieldElement(n))
    }

    pub fn to_bytes_be(&self) -> [u8; 32] {
        let v = self.0.to_bytes_be(32);
        let mut out = [0u8; 32];
        let start = if v.len() >= 32 { v.len() - 32 } else { 0 };
        let copy_len = v.len().min(32);
        let dst_start = 32 - copy_len;
        out[dst_start..].copy_from_slice(&v[start..start + copy_len]);
        out
    }
}
```

```rust
        let sum = self.0.add(&other.0);
        let p = p_modulus();
        if sum.cmp(&p) != std::cmp::Ordering::Less {
            FieldElement(sum.checked_sub(&p).unwrap_or(BigUint::zero()))
        } else {
            FieldElement(sum)
        }
    }

    pub fn sub(&self, other: &Self) -> Self {
        match self.0.checked_sub(&other.0) {
            Some(d) => FieldElement(d),
            None => {
                // self < other: return (self + p) - other
                let p = p_modulus();
                let a_plus_p = self.0.add(&p);
                FieldElement(a_plus_p.checked_sub(&other.0).unwrap_or(BigUint::zero()))
            }
        }
    }

    pub fn neg(&self) -> Self {
        if self.0.is_zero() {
            FieldElement::zero()
        } else {
            FieldElement(p_modulus().checked_sub(&self.0).unwrap_or(BigUint::zero()))
        }
    }

    pub fn mul(&self, other: &Self) -> Self {
        let product = self.0.mul(&other.0);
        // Reduce mod p via div_rem (simple; performance adequate).
        let (_, r) = product
            .div_rem(&p_modulus())
            .unwrap_or((BigUint::zero(), BigUint::zero()));
        FieldElement(r)
    }

        self.mul(self)
```

```rust
    /// Multiplicative inverse via Fermat's little theorem: a^(p-2) mod p.
    /// Returns `None` if `self` is zero.
    pub fn invert(&self) -> Option<Self> {
        if self.0.is_zero() {
            return None;

        // p - 2
        let two = BigUint::from_u64(2);
        let exp = p.checked_sub(&two)?;
        let inv = self.0.modexp(&exp, &p)?;
        Some(FieldElement(inv))
```

```rust
impl Point {
    pub fn infinity() -> Self {
        Point {
            x: FieldElement::one(),
            y: FieldElement::one(),
            z: FieldElement::zero(),
        }
    }

    pub fn generator() -> Self {
        Point {
            x: FieldElement(gx()),
            y: FieldElement(gy()),
            z: FieldElement::one(),
        }
    }

    pub fn is_infinity(&self) -> bool {
        self.z.0.is_zero()
    }

    /// True if the point satisfies y²·Z⁶ = x³·Z² + a·x·Z⁴·Z² + b·Z⁶ (Jacobian).
    /// For affine points (Z=1) this is the standard y² = x³ − 3x + b.
    pub fn is_on_curve(&self) -> bool {
        if self.is_infinity() {
            return true;
        }
        // For Z=1 only (our entry points from_uncompressed and generator).
        let one = FieldElement::one();
        if self.z != one {
            // General Jacobian check can be added later; verify-only path
            // only needs Z=1 at the start and converts back to affine at end.
            return true; // assume caller maintains invariants post-operations
        }
        let x = &self.x;
        let y = &self.y;
        let lhs = y.square();
        let x2 = x.square();
        let x3 = x2.mul(x);
        // a = -3, so a·x = -3·x = -(x+x+x)
        let three_x = x.add(x).add(x);
        let b = FieldElement(b_coeff());
        let rhs = x3.sub(&three_x).add(&b);
        lhs == rhs
    }

    /// Parse 65-byte uncompressed point: 0x04 || x || y.
    pub fn from_uncompressed(bytes: &[u8; 65]) -> crate::Result<Self> {
        if bytes[0] != 0x04 {
            return Err(Error::Tls(format!(
                "P-256 point: expected 0x04 prefix, got 0x{:02x}",
                bytes[0]
            )));
        }
        let mut x_bytes = [0u8; 32];
        x_bytes.copy_from_slice(&bytes[1..33]);
        let mut y_bytes = [0u8; 32];
        y_bytes.copy_from_slice(&bytes[33..65]);
        let x = FieldElement::from_bytes_be(&x_bytes)
            .ok_or_else(|| Error::Tls("P-256 point: x ≥ p".to_string()))?;
        let y = FieldElement::from_bytes_be(&y_bytes)
            .ok_or_else(|| Error::Tls("P-256 point: y ≥ p".to_string()))?;
        let p = Point {
            x,
            y,
            z: FieldElement::one(),
        };
        if !p.is_on_curve() {
            return Err(Error::Tls("P-256 point: not on curve".to_string()));
        }
        Ok(p)
    }
}
```

```rust
    /// Convert Jacobian to affine x-coordinate only. Returns None for infinity.
    pub fn affine_x_bytes(&self) -> Option<[u8; 32]> {
        if self.is_infinity() {
            return None;
        }
        // x_affine = X / Z²
        let z2 = self.z.square();
        let z2_inv = z2.invert()?;
        let x_aff = self.x.mul(&z2_inv);
        Some(x_aff.to_bytes_be())
    }

    /// Point doubling in Jacobian coordinates.
    /// Formulas: https://www.hyperelliptic.org/EFD/g1p/auto-shortw-jacobian-3.html
    /// Using a = -3 specialization.
    pub fn double(&self) -> Self {
        if self.is_infinity() || self.y.0.is_zero() {
            return Point::infinity();
        }
        let x1 = &self.x;
        let y1 = &self.y;
        let z1 = &self.z;
        // delta = Z1²
        let delta = z1.square();
        // gamma = Y1²
        let gamma = y1.square();
        // beta = X1 * gamma
        let beta = x1.mul(&gamma);
        // alpha = 3 * (X1 - delta) * (X1 + delta)
        let x1_minus_delta = x1.sub(&delta);
        let x1_plus_delta = x1.add(&delta);
        let t = x1_minus_delta.mul(&x1_plus_delta);
        let alpha = t.add(&t).add(&t); // 3*t
        // X3 = alpha² - 8*beta
        let alpha2 = alpha.square();
        let eight_beta = beta.add(&beta).add(&beta).add(&beta).add(&beta).add(&beta).add(&beta).add(&beta);
        let x3 = alpha2.sub(&eight_beta);
        // Z3 = (Y1 + Z1)² - gamma - delta
        let y1_plus_z1 = y1.add(z1);
        let z3 = y1_plus_z1.square().sub(&gamma).sub(&delta);
        // Y3 = alpha * (4*beta - X3) - 8*gamma²
        let four_beta = beta.add(&beta).add(&beta).add(&beta);
        let gamma2 = gamma.square();
        let eight_gamma2 = gamma2.add(&gamma2).add(&gamma2).add(&gamma2).add(&gamma2).add(&gamma2).add(&gamma2).add(&gamma2);
        let y3 = alpha.mul(&four_beta.sub(&x3)).sub(&eight_gamma2);
        Point { x: x3, y: y3, z: z3 }
    }

    /// Point addition in Jacobian coordinates.
    /// Handles infinity operands and the point-doubling fallback.
    pub fn add(&self, other: &Self) -> Self {
        if self.is_infinity() {
            return other.clone();
        }
        if other.is_infinity() {
            return self.clone();
        }
        // Formulas: add-2007-bl (add-2007-bl).
        let x1 = &self.x;
        let y1 = &self.y;
        let z1 = &self.z;
        let x2 = &other.x;
        let y2 = &other.y;
        let z2 = &other.z;
        // z1z1 = Z1²
        let z1z1 = z1.square();
        let z2z2 = z2.square();
        // U1 = X1 * Z2²; U2 = X2 * Z1²
        let u1 = x1.mul(&z2z2);
        let u2 = x2.mul(&z1z1);
        // S1 = Y1 * Z2³; S2 = Y2 * Z1³
        let s1 = y1.mul(&z2.mul(&z2z2));
        let s2 = y2.mul(&z1.mul(&z1z1));

        if u1 == u2 {
            if s1 == s2 {
                return self.double();
            }
            return Point::infinity();
        }

        let h = u2.sub(&u1);
        let i = h.add(&h).square();
        let j = h.mul(&i);
        let r = s2.sub(&s1).add(&s2.sub(&s1)); // 2*(S2-S1)
        let v = u1.mul(&i);
        // X3 = r² - J - 2V
        let two_v = v.add(&v);
        let x3 = r.square().sub(&j).sub(&two_v);
        // Y3 = r*(V - X3) - 2*S1*J
        let s1_j = s1.mul(&j);
        let two_s1_j = s1_j.add(&s1_j);
        let y3 = r.mul(&v.sub(&x3)).sub(&two_s1_j);
        // Z3 = ((Z1+Z2)² - Z1Z1 - Z2Z2) * H
        let z1_plus_z2 = z1.add(z2);
        let z3 = z1_plus_z2.square().sub(&z1z1).sub(&z2z2).mul(&h);
        Point { x: x3, y: y3, z: z3 }
    }
```

```rust
    /// Scalar is 32 big-endian bytes treated as an integer.
    /// Not constant-time — for verification only.
    pub fn scalar_mul(&self, scalar: &[u8; 32]) -> Self {
        let mut result = Point::infinity();
        let mut seen_one = false;
        for byte in scalar.iter() {
            for bit_pos in (0..8).rev() {
                if seen_one {
                    result = result.double();
                }
                if (byte >> bit_pos) & 1 == 1 {
                    if seen_one {
                        result = result.add(self);
                    } else {
                        result = self.clone();
                        seen_one = true;
                    }
                }
            }
        }
        result
    }
```

```rust
//! ECDSA signature verification over the P-256 curve, SHA-256 digest.

#![allow(dead_code)]

use crate::Error;
use crate::core::net::tls::p256::Point;

pub struct EcdsaPublicKey {
    pub point: Point,
}

/// OID 1.2.840.10045.2.1 id-ecPublicKey.
const OID_EC_PUBLIC_KEY: &[u8] = &[0x2a, 0x86, 0x48, 0xce, 0x3d, 0x02, 0x01];
/// OID 1.2.840.10045.3.1.7 prime256v1 / secp256r1.
const OID_SECP256R1: &[u8] = &[0x2a, 0x86, 0x48, 0xce, 0x3d, 0x03, 0x01, 0x07];

impl EcdsaPublicKey {
    /// Parse SubjectPublicKeyInfo DER for an EC public key on P-256.
    pub fn from_spki(der: &[u8]) -> crate::Result<Self> {
        use crate::core::asn1::Parser;
        let mut top = Parser::new(der);
        let mut spki = top
            .read_sequence()
            .map_err(|e| Error::Tls(format!("ECDSA SPKI: {e}")))?;

        // AlgorithmIdentifier: OID id-ecPublicKey + OID namedCurve
        let mut alg = spki
            .read_sequence()
            .map_err(|e| Error::Tls(format!("ECDSA alg: {e}")))?;
        let oid = alg
            .read_oid()
            .map_err(|e| Error::Tls(format!("ECDSA alg OID: {e}")))?;
        if oid != OID_EC_PUBLIC_KEY {
            return Err(Error::Tls(format!(
                "ECDSA SPKI: expected id-ecPublicKey, got {}-byte OID",
                oid.len()
            )));
        }
        let curve_oid = alg
            .read_oid()
            .map_err(|e| Error::Tls(format!("ECDSA curve OID: {e}")))?;
        if curve_oid != OID_SECP256R1 {
            return Err(Error::Tls(format!(
                "ECDSA SPKI: expected secp256r1, got {}-byte OID",
                curve_oid.len()
            )));
        }
        while !alg.is_empty() {
            let _ = alg
                .read_any()
                .map_err(|e| Error::Tls(format!("ECDSA alg extra: {e}")))?;
        }

        // subjectPublicKey BIT STRING = 0x04 || x(32) || y(32)
        let bits = spki
            .read_bit_string()
            .map_err(|e| Error::Tls(format!("ECDSA bit string: {e}")))?;
        if bits.unused_bits != 0 {
            return Err(Error::Tls(format!(
                "ECDSA SPKI: BIT STRING unused_bits = {}",
                bits.unused_bits
            )));
        }
        if bits.bytes.len() != 65 {
            return Err(Error::Tls(format!(
                "ECDSA SPKI: expected 65-byte point, got {}",
                bits.bytes.len()
            )));
        }
        let mut point_bytes = [0u8; 65];
        point_bytes.copy_from_slice(bits.bytes);
        let point = Point::from_uncompressed(&point_bytes)?;
        Ok(EcdsaPublicKey { point })
    }
}
```

```rust
/// Verify ECDSA-SHA256 signature with a P-256 public key.
/// `signature` is DER `SEQUENCE { r INTEGER, s INTEGER }`.
pub fn verify_ecdsa_sha256(
    pk: &EcdsaPublicKey,
    msg: &[u8],
    signature: &[u8],
) -> crate::Result<()> {
    use crate::core::net::tls::crypto::sha256::Sha256;
    let digest = Sha256::hash(msg);
    verify_ecdsa_sha256_prehashed(pk, &digest, signature)
}

/// Same as `verify_ecdsa_sha256` but accepts a precomputed SHA-256 digest.
pub fn verify_ecdsa_sha256_prehashed(
    pk: &EcdsaPublicKey,
    digest: &[u8; 32],
    signature: &[u8],
) -> crate::Result<()> {
    use crate::core::asn1::Parser;
    use crate::core::bigint::BigUint;
    use crate::core::net::tls::p256::{Point, n_order};

    // 1. DER-decode signature into r, s BigUints.
    let mut parser = Parser::new(signature);
    let mut seq = parser
        .read_sequence()
        .map_err(|e| Error::Tls(format!("ECDSA sig seq: {e}")))?;
    let r_bytes = seq
        .read_integer()
        .map_err(|e| Error::Tls(format!("ECDSA r: {e}")))?;
    let s_bytes = seq
        .read_integer()
        .map_err(|e| Error::Tls(format!("ECDSA s: {e}")))?;
    seq.finish()
        .map_err(|e| Error::Tls(format!("ECDSA sig trailing: {e}")))?;
    let r = BigUint::from_bytes_be(r_bytes);
    let s = BigUint::from_bytes_be(s_bytes);

    // 2. Range check: 1 ≤ r, s < n.
    let n = n_order();
    if r.is_zero() || r.cmp(&n) != std::cmp::Ordering::Less {
        return Err(Error::Tls("ECDSA: r out of range".to_string()));
    }
    if s.is_zero() || s.cmp(&n) != std::cmp::Ordering::Less {
        return Err(Error::Tls("ECDSA: s out of range".to_string()));
    }

    // 3. e = digest as BigUint (SHA-256 = 256 bits = n bit-width, no truncation).
    let e = BigUint::from_bytes_be(digest);

    // 4. w = s^-1 mod n
    let w = s
        .mod_inverse(&n)
        .ok_or_else(|| Error::Tls("ECDSA: s has no inverse mod n".to_string()))?;

    // 5. u1 = e*w mod n, u2 = r*w mod n
    let (_, u1) = e
        .mul(&w)
        .div_rem(&n)
        .ok_or_else(|| Error::Tls("ECDSA: u1 reduction failed".to_string()))?;
    let (_, u2) = r
        .mul(&w)
        .div_rem(&n)
        .ok_or_else(|| Error::Tls("ECDSA: u2 reduction failed".to_string()))?;

    // 6. P = u1·G + u2·Q
    let u1_bytes_vec = u1.to_bytes_be(32);
    let mut u1_bytes = [0u8; 32];
    let src_start = u1_bytes_vec.len().saturating_sub(32);
    let dst_start = 32usize.saturating_sub(u1_bytes_vec.len());
    let copy_len = u1_bytes_vec.len().min(32);
    u1_bytes[dst_start..dst_start + copy_len]
        .copy_from_slice(&u1_bytes_vec[src_start..src_start + copy_len]);

    let u2_bytes_vec = u2.to_bytes_be(32);
    let mut u2_bytes = [0u8; 32];
    let src_start = u2_bytes_vec.len().saturating_sub(32);
    let dst_start = 32usize.saturating_sub(u2_bytes_vec.len());
    let copy_len = u2_bytes_vec.len().min(32);
    u2_bytes[dst_start..dst_start + copy_len]
        .copy_from_slice(&u2_bytes_vec[src_start..src_start + copy_len]);

    let g = Point::generator();
    let u1g = g.scalar_mul(&u1_bytes);
    let u2q = pk.point.scalar_mul(&u2_bytes);
    let sum = u1g.add(&u2q);
    if sum.is_infinity() {
        return Err(Error::Tls("ECDSA: u1·G + u2·Q = infinity".to_string()));
    }

    // 7. Compare x_P mod n with r.
    let x_bytes = sum
        .affine_x_bytes()
        .ok_or_else(|| Error::Tls("ECDSA: failed to extract affine x".to_string()))?;
    let x_big = BigUint::from_bytes_be(&x_bytes);
    let (_, v) = x_big
        .div_rem(&n)
        .ok_or_else(|| Error::Tls("ECDSA: x mod n failed".to_string()))?;
    if v == r {
        Ok(())
    } else {
        Err(Error::Tls("ECDSA: signature mismatch".to_string()))
    }
}
```
