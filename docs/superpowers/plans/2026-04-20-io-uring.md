# io_uring Async Reactor Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the global epoll async reactor with an io_uring POLL_ADD reactor, eliminating the DEL+ADD syscall overhead per wake cycle.

**Architecture:** `IoUringReactor` uses `IORING_OP_POLL_ADD` (readiness-based, compatible with the existing `Future::poll` interface) via manually managed SQ/CQ ring buffers using raw FFI syscalls. The terminal `EventLoop` keeps its own `EpollReactor` instance because it uses a synchronous token-based API (`register_fd`/`wait_tokens`) that doesn't map to io_uring's one-shot model. A new `PlatformAsyncReactor` type alias in `platform/mod.rs` switches the global `runtime/reactor.rs` from epoll to io_uring on Linux.

**Tech Stack:** Raw Linux io_uring FFI (syscalls 425/426), `mmap` for ring buffers, zero external deps.

## Key Design Facts

### io_uring kernel interface

- `SYS_IO_URING_SETUP = 425`, `SYS_IO_URING_ENTER = 426`, `SYS_MMAP = 9`, `SYS_MUNMAP = 11`, `SYS_CLOSE = 3` (x86_64)
- `io_uring_setup(32, &mut params)` → ring fd. Kernel fills `params.sq_off`/`cq_off` (field byte offsets within mmaps) and `params.features`.
- Three mmaps: SQ ring at `IORING_OFF_SQ_RING=0`, CQ ring at `IORING_OFF_CQ_RING=0x0800_0000` (unless `IORING_FEAT_SINGLE_MMAP`, then same as SQ), SQEs at `IORING_OFF_SQES=0x1000_0000`.
- SQ ring mmap size: `sq_off.array + sq_entries * 4`
- CQ ring mmap size: `cq_off.cqes + cq_entries * 16`
- SQE array mmap size: `sq_entries * 64`

### Submitting an SQE

1. Read kernel's `*sq_head` (acquire) — tells us how many slots the kernel has consumed
2. Check not full: `sq_tail_local - *sq_head < sq_entries`
3. `idx = sq_tail_local & sq_mask`
4. Write SQE into `sqes[idx]`
5. Write `sq_array[idx] = idx` (identity mapping — sets which SQE slot this ring position uses)
6. Increment `sq_tail_local`
7. On `poll()`: fence(Release), then write `sq_tail_local` → `*sq_ktail`; call `io_uring_enter`

### Draining the CQ

```
head = *cq_head (relaxed)
fence(Acquire)
tail = *cq_tail (relaxed)
while head != tail:
    cqe = cq_cqes[head & cq_mask]
    process cqe.user_data
    head = head.wrapping_add(1)
fence(Release)
*cq_head = head (relaxed)
```

### POLL_ADD semantics

- One-shot: fires once when the fd becomes readable/writable, then is removed automatically.
- `user_data` in CQE equals `user_data` in SQE — use this as token to look up Waker.
- `POLLIN | POLLHUP | POLLERR` for reads; `POLLOUT | POLLERR` for writes.

### Timeout

- Submit `IORING_OP_TIMEOUT` SQE with `addr = &KernelTimespec`, `len = 1`, `off = 0` (pure time-based).
- `io_uring_enter` with `min_complete=1` waits for one CQE (real event OR timeout CQE).
- When timeout fires, `io_uring_enter` returns `-ETIME` (errno 62). Treat this as success.
- Timeout CQE has `user_data = 0`; since tokens start at 1, it's silently ignored in drain.

### POLL_REMOVE for deregister

- Submit `IORING_OP_POLL_REMOVE` with `addr = original_token` (cancels in-flight POLL_ADD).
- Waker is removed from HashMap before submitting POLL_REMOVE, so the resulting cancel CQE is ignored.
- POLL_REMOVE is submitted lazily (pending until next `poll()`).

### Struct sizes (must be exact)

- `Sqe`: 64 bytes. Layout: `u8 + u8 + u16 + i32 + u64 + u64 + u32 + u32 + u64 + [u64; 3]` = 1+1+2+4+8+8+4+4+8+24 = 64 ✓
- `Cqe`: 16 bytes. Layout: `u64 + i32 + u32` = 8+4+4 = 16 ✓
- Verified at compile time with `const _: () = assert!(size_of::<Sqe>() == 64)`.

```rust
unsafe fn do_mmap(ring_fd: i32, offset: i64, size: usize) -> crate::Result<*mut u8> {

        syscall(
            SYS_MMAP,
            0i64,
            size as i64,
            (PROT_READ | PROT_WRITE) as i64,
            MAP_SHARED as i64 | MAP_POPULATE as i64,
            ring_fd as i64,
            offset,

    Ok(ptr as *mut u8)
```

```rust
// Append to src/core/platform/unix/io_uring.rs

pub struct IoUringReactor {
    ring_fd: i32,

    // SQ ring (mmap)
    sq_ptr:     *mut u8,
    sq_ring_sz: usize,
    sq_ktail:   *mut u32,   // kernel-shared tail (we write here to submit)
    sq_khead:   *const u32, // kernel-shared head (kernel advances as it consumes)
    sq_mask:    u32,
    sq_array:   *mut u32,

    // SQE array (separate mmap)
    sqes:    *mut Sqe,
    sqes_sz: usize,

    // CQ ring (may share mmap with SQ)
    cq_ptr:    *mut u8,
    cq_ring_sz: usize,
    cq_own:    bool, // true = separate mmap, must be munmap'd independently
    cq_ktail:  *const u32,
    cq_khead:  *mut u32,
    cq_mask:   u32,
    cq_cqes:   *const Cqe,

    // Submission tracking (local, not shared with kernel)
    sq_tail:  u32, // next SQE index to fill
    pending:  u32, // SQEs written but not yet submitted via io_uring_enter

    // Waker registry
    wakers:     HashMap<u64, Waker>,
    next_token: u64,

    // Reusable timeout storage (addr must outlive the SQE until io_uring_enter)
    ts: KernelTimespec,
}

unsafe impl Send for IoUringReactor {}

impl IoUringReactor {
    pub fn new() -> crate::Result<Self> {
        let mut params = IoUringParams::zeroed();

        let ring_fd = unsafe {
            let ret = syscall(SYS_IO_URING_SETUP, 32i64, &mut params as *mut IoUringParams as i64);
            if ret < 0 { return Err(io_err(ret)); }
            ret as i32
        };

        let sq_entries = params.sq_entries as usize;
        let cq_entries = params.cq_entries as usize;

        let sq_ring_sz = params.sq_off.array as usize + sq_entries * 4;
        let cq_ring_sz = params.cq_off.cqes as usize + cq_entries * 16;
        let sqes_sz    = sq_entries * 64;

        let sq_ptr = unsafe { do_mmap(ring_fd, IORING_OFF_SQ_RING, sq_ring_sz)? };

        let (cq_ptr, cq_ring_sz_actual, cq_own) =
            if params.features & IORING_FEAT_SINGLE_MMAP != 0 {
                (sq_ptr, cq_ring_sz, false)
            } else {
                let p = unsafe { do_mmap(ring_fd, IORING_OFF_CQ_RING, cq_ring_sz)? };
                (p, cq_ring_sz, true)
            };

        let sqes = unsafe { do_mmap(ring_fd, IORING_OFF_SQES, sqes_sz)? as *mut Sqe };

        // Resolve pointers into the ring mmaps using kernel-provided byte offsets.
        let sq_ktail = unsafe { sq_ptr.add(params.sq_off.tail as usize)      as *mut u32  };
        let sq_khead = unsafe { sq_ptr.add(params.sq_off.head as usize)      as *const u32};
        let sq_mask  = unsafe { *(sq_ptr.add(params.sq_off.ring_mask as usize) as *const u32) };
        let sq_array = unsafe { sq_ptr.add(params.sq_off.array as usize)     as *mut u32  };

        let cq_ktail = unsafe { cq_ptr.add(params.cq_off.tail as usize)      as *const u32};
        let cq_khead = unsafe { cq_ptr.add(params.cq_off.head as usize)      as *mut u32  };
        let cq_mask  = unsafe { *(cq_ptr.add(params.cq_off.ring_mask as usize) as *const u32) };
        let cq_cqes  = unsafe { cq_ptr.add(params.cq_off.cqes as usize)      as *const Cqe};

        Ok(Self {
            ring_fd,
            sq_ptr, sq_ring_sz, sq_ktail, sq_khead, sq_mask, sq_array,
            sqes, sqes_sz,
            cq_ptr, cq_ring_sz: cq_ring_sz_actual, cq_own,
            cq_ktail, cq_khead, cq_mask, cq_cqes,
            sq_tail: 0,
            pending: 0,
            wakers: HashMap::new(),
            next_token: 1,
            ts: KernelTimespec { tv_sec: 0, tv_nsec: 0 },
        })
    }
}

impl Drop for IoUringReactor {
    fn drop(&mut self) {
        unsafe {
            syscall(SYS_MUNMAP, self.sq_ptr as i64, self.sq_ring_sz as i64);
            if self.cq_own {
                syscall(SYS_MUNMAP, self.cq_ptr as i64, self.cq_ring_sz as i64);
            }
            syscall(SYS_MUNMAP, self.sqes as i64, self.sqes_sz as i64);
            syscall(SYS_CLOSE, self.ring_fd as i64);
        }
    }
}
```

```rust
impl IoUringReactor {
    /// Returns a mutable reference to the next available SQE slot, or None if SQ is full.
    fn get_sqe(&mut self) -> Option<&mut Sqe> {
        // sq_khead is the kernel's consumed index; if tail - head >= entries, ring is full.
        let head = unsafe { (self.sq_khead as *const AtomicU32).as_ref().unwrap().load(Ordering::Acquire) };
        if self.sq_tail.wrapping_sub(head) >= self.sq_entries() {
            return None;
        }
        let idx = self.sq_tail & self.sq_mask;
        // Identity mapping: ring position idx uses SQE slot idx.
        unsafe { *self.sq_array.add(idx as usize) = idx; }
        self.sq_tail = self.sq_tail.wrapping_add(1);
        self.pending += 1;
        unsafe { Some(&mut *self.sqes.add(idx as usize)) }
    }

    fn sq_entries(&self) -> u32 {
        self.sq_mask + 1
    }

    /// Flush pending SQEs to the kernel ring tail and call io_uring_enter.
    /// Waits for at least one CQE (real event or timeout). Returns count of real events.
    pub fn poll(&mut self, timeout: Duration) -> crate::Result<usize> {
        // Submit a TIMEOUT SQE so io_uring_enter doesn't block indefinitely.
        self.ts.tv_sec  = timeout.as_secs() as i64;
        self.ts.tv_nsec = timeout.subsec_nanos() as i64;
        let ts_addr = &self.ts as *const KernelTimespec as u64;

        if let Some(sqe) = self.get_sqe() {
            *sqe = Sqe {
                opcode: IORING_OP_TIMEOUT,
                flags: 0, ioprio: 0, fd: -1,
                off: 0,         // off=0: pure time-based timeout
                addr: ts_addr,
                len: 1,
                op_flags: 0,
                user_data: 0,   // timeout CQE always has user_data=0, ignored in drain
                _pad: [0; 3],
            };
        }

        let to_submit = self.pending;
        self.pending = 0;

        // Flush: make SQEs visible to the kernel.
        fence(Ordering::Release);
        unsafe {
            (self.sq_ktail as *mut AtomicU32)
                .as_mut().unwrap()
                .store(self.sq_tail, Ordering::Relaxed);
        }

        // Submit + wait for at least 1 completion (real event or timeout).
        let ret = unsafe {
            syscall(
                SYS_IO_URING_ENTER,
                self.ring_fd as i64,
                to_submit as i64,
                1i64,                           // min_complete
                IORING_ENTER_GETEVENTS as i64,
                0i64,                           // sigmask = NULL
                0i64,                           // _nsig = 0
            )
        };
        if ret < 0 && -ret != ERRNO_ETIME as i64 {
            return Err(io_err(ret));
        }

        self.drain_cq()
    }

    fn drain_cq(&mut self) -> crate::Result<usize> {
        let mut count = 0usize;
        loop {
            let head = unsafe {
                (self.cq_khead as *mut AtomicU32).as_mut().unwrap().load(Ordering::Relaxed)
            };
            fence(Ordering::Acquire);
            let tail = unsafe {
                (self.cq_ktail as *const AtomicU32).as_ref().unwrap().load(Ordering::Relaxed)
            };
            if head == tail { break; }

            let idx = head & self.cq_mask;
            let cqe = unsafe { &*self.cq_cqes.add(idx as usize) };
            let token = cqe.user_data;

            if let Some(waker) = self.wakers.remove(&token) {
                waker.wake();
                count += 1;
            }
            // token == 0 → timeout CQE, silently ignored (no waker stored for 0)

            fence(Ordering::Release);
            unsafe {
                (self.cq_khead as *mut AtomicU32)
                    .as_mut().unwrap()
                    .store(head.wrapping_add(1), Ordering::Relaxed);
            }
        }
        Ok(count)
    }

    pub fn register_read(&mut self, handle: RawHandle, waker: Waker) -> crate::Result<u64> {
        let token = self.next_token;
        self.next_token += 1;
        let sqe = self.get_sqe()
            .ok_or_else(|| crate::Error::Io(std::io::Error::other("io_uring SQ full")))?;
        *sqe = Sqe {
            opcode: IORING_OP_POLL_ADD,
            flags: 0, ioprio: 0,
            fd: handle,
            off: 0,
            addr: 0,
            len: 0,
            op_flags: POLLIN | POLLHUP | POLLERR,
            user_data: token,
            _pad: [0; 3],
        };
        self.wakers.insert(token, waker);
        Ok(token)
    }

    pub fn register_write(&mut self, handle: RawHandle, waker: Waker) -> crate::Result<u64> {
        let token = self.next_token;
        self.next_token += 1;
        let sqe = self.get_sqe()
            .ok_or_else(|| crate::Error::Io(std::io::Error::other("io_uring SQ full")))?;
        *sqe = Sqe {
            opcode: IORING_OP_POLL_ADD,
            flags: 0, ioprio: 0,
            fd: handle,
            off: 0,
            addr: 0,
            len: 0,
            op_flags: POLLOUT | POLLERR,
            user_data: token,
            _pad: [0; 3],
        };
        self.wakers.insert(token, waker);
        Ok(token)
    }

    pub fn deregister(&mut self, token: u64) -> crate::Result<()> {
        self.wakers.remove(&token);
        // Submit POLL_REMOVE to cancel the in-flight POLL_ADD (if still pending).
        // This is best-effort: if the SQ is full or already fired, the cancellation is a no-op.
        if let Some(sqe) = self.get_sqe() {
            *sqe = Sqe {
                opcode: IORING_OP_POLL_REMOVE,
                flags: 0, ioprio: 0, fd: 0,
                off: 0,
                addr: token, // addr = user_data of the SQE to cancel
                len: 0,
                op_flags: 0,
                user_data: 0, // don't care about this completion
                _pad: [0; 3],
            };
        }
        Ok(())
    }
}
```
